Answering questions

1. How does javascript work?

Answer: JavaScript is a client-side scripting language. It runs in the browser using a JavaScript engine. Chrome uses V8 engine. It turns the JavaScript code into a line-by-line interpreted language. To make it an interpreted language it uses a Just In Time Compiler inside it. It only interests the code every time the method is invoked. Ultimately it turns the JavaScript code into ByteCode. The ByteCode will become machine code and then the computer can read the code.

2. What does javascript event loop do?

Answer: Javascript is a single-threaded programing language. Javascript uses stack to manage the function call. Javascript can run one statement at a time. So it runs the first statement then puts that in the stack and when the code runs completely it will pop the statement and then go to the second one. But if some statement is a bock code, javascript will put the statements in Web APIis. Then the statements will go in a task queue and When the stack is empty the event loop pushes the statements inside the stack one by one and run the code.
